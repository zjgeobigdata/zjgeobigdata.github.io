# 分割虚线

分割虚线

## 基础用法

<DashDivider />

```vue
<template>
  <dz-dashDivider/>
</template>
```

## Api

| 参数        | 说明     | 类型       | 默认值 |
| --------- | ------ | -------- | --- |
| type      | 分割线方向   | 'horizontal' 或 'vertical' | 'horizontal'   |
| bgColor | 背景颜色  | string | 'red'   |
| gradientColorStart | 线条渐变色起始色  | string | 'blue'   |
| gradientColorEnd    | 线条渐变色终止色 | string   | 'green'   |
